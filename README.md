**My Counter**

A simple web counter application built with HTML, CSS, and vanilla JavaScript as a project for the Start2Impact course.

**Overview**

The app allows you to:

* increment the counter (+)
* decrement the counter (-)
* reset the counter (Reset)

The interface is dynamically generated via JavaScript and features a modern design with gradients and subtle CSS animations.

**Demo / Project Structure**
